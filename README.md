# 🔒 מפעיל סיסמאות אקראיות

פרויקט פשוט ליצירת סיסמאות אקראיות וחזקות.

## 📋 תיאור

אפליקציית Web פשוטה המאפשרת ליצור סיסמאות אקראיות עם אפשרויות התאמה אישית:
- בחירת אורך הסיסמה (4-32 תווים)
- כללת אותיות קטנות (a-z)
- כללת אותיות גדולות (A-Z)
- כללת מספרים (0-9)
- כללת סימנים מיוחדים (@#$%!)

## 🚀 איך להשתמש

1. פתח את הקובץ `index.html` בדפדפן
2. בחר את אורך הסיסמה באמצעות המחוון
3. סמן/בטל סימון את סוגי התווים הרצויים
4. לחץ על "יצירת סיסמה"
5. הסיסמה החדשה תוצג בשדה העליון

## 📁 קבצים

- `index.html` - מבנה הדף
- `style.css` - עיצוב וממשק משתמש
- `main.js` - לוגיקת יצירת הסיסמאות

## 💡 קונספטים שנלמדו

הפרויקט מדגים:
- ✅ שימוש ב-DOM API
- ✅ טיפול באירועים (Event Handling)
- ✅ לוגיקה בוליאנית
- ✅ לולאות ואקראיות
- ✅ קבועים (`const`) ומשתנים (`let`)
- ✅ פונקציות
- ✅ מניפולציה של מחרוזות

## 🛠️ טכנולוגיות

- HTML5
- CSS3
- JavaScript (Vanilla)
